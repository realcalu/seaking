# seaking

从403条渣男对话语录中蒸馏的情感操控模式识别与防御分析Skill，并融合「深情祖师爷」童锦程式套路库：反舔狗、反测试、重真诚、看行动，也拆开场、台阶、邀约和反馈。

它的用法很直接：帮你同时看清三件事：
- 对方是不是在用话术操控你
- 你自己是不是正在用讨好、试探、冷处理把关系拖进消耗
- 一段追求/搭讪/邀约到底卡在开场、台阶、反馈还是投入节奏

## 祖师爷部分

本Skill借鉴了本地 `tong-jincheng-skill` 中的童锦程关系洞察。这里的「祖师爷」部分有两层：一层是健康关系对照组，用来判断投入、边界和真诚；另一层是套路库，用来拆开场、台阶、邀约和反馈。

吸收的核心判断：
- **吸引力 > 讨好**：用于识别不对等投入和自我消耗
- **人性不可考验**：用于提醒用户不要用失联、冷处理、假分手测试关系
- **真诚沟通**：用于把需求说清楚，把承诺落到可验证行动
- **给台阶**：用于降低尴尬，让对话和邀约有自然继续的理由
- **看行动，不看人设**：对方说自己真诚、专一、会改，都要回到时间、频率、透明度、边界和复发率
- **轻开场**：先让对话成立，再看对方是否接话、延展和反问
- **低成本邀约**：用具体、轻量、可改期的方式测试线下意愿
- **反馈校准**：把"改天""下次""最近忙"放回具体行动里判断
