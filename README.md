这是一个 Claude Code Skill，放置于 `~/.claude/skills/seaking/` 即可激活。

- 话术分析
- 情感操控
- gaslighting
- PUA识别

## 核心框架

| 组件 | 数量 | 说明 |
|------|------|------|
| 心智模型 | 6个 | 情感成本最小化、责任外化/反转归因、模糊性武器化、阶段性投入调控、三角关系稀释、健康关系对照组 |
| 决策启发式 | 11条 | 失联借口、空头承诺、反转指责、关系最小化、假道歉、被动分手、紧急安抚、终极免责、讨好型投入止损、反测试、行为闭环验证 |
| 关系阶段 | 5阶段 | 热恋→冷却→试探→操控→撤离 |
| 表达DNA | 11维度 | 句式、词汇、节奏、标签、承诺模板、自我标签、测试冲动等 |

## 融合说明

本Skill借鉴了本地 `tong-jincheng-skill` 中的部分关系洞察，但只做防御型吸收：
- **吸引力 > 讨好**：用于识别不对等投入和自我消耗
- **人性不可考验**：用于提醒用户不要用失联、冷处理、假分手测试关系
- **真诚沟通**：用于把需求说清楚，把承诺落到可验证行动
- **给台阶**：只用于体面沟通和修正空间，不用于诱导或操控对方

## 生成方式

本Skill由 [女娲 · Skill造人术](https://github.com/alchaincyf/nuwa-skill) 从403条用户提供的语录中蒸馏生成。

特别感谢 zx 提供原始语料与灵感来源，让这个情感操控话术识别框架得以被整理、提炼和复用。

蒸馏流程：
1. **Phase 0**: 本地语料模式确认（用户提供403条语录）
2. **Phase 1**: 6维度并行分析（著作/对话/表达/他者/决策/时间线）
3. **Phase 2**: 框架提炼（三重验证筛选心智模型）
4. **Phase 3**: Skill构建
5. **Phase 4**: 质量验证（已知测试/边缘测试/风格测试/结构检查 — 全部通过）
